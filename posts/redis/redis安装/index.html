Redis安装

1. CentOS

1.1 下载redis源码并编译

# 下载源码压缩包 具体url可以去官网看
wget https://download.redis.io/releases/redis-5.0.10.tar.gz
# 解压
tar zxf redis-5.0.10.tar.gz
# 最好将解压后的文件夹移动到一个合理的位置 如/usr/local
# 进入源码文件夹
cd redis-5.0.10
# 编译
make
# 进入src目录并make install
cd ./src
make install
